# nodejs-app


To run project please follow these steps:
  1. npm install
  2. sequelize-cli db:migrate 
  3. node index.js
Note: you are needed ./config/config.json in root of your folder in order to run sequlize migrations.

ENV template:
NODE_ENV=
DB_HOST=
DB_USERNAME=
DB_PASSWORD=
DB_NAME=
DB_DIALECT=
DB_POOL_MAX=
DB_POOL_MIN=
DB_POOL_ACQUIRE=
DB_POOL_IDLE=
AUTH_SECRET=

  
Happy coding :)
